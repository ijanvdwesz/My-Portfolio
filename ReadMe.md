# Ijan Van der Westhuizen - Portfolio

This is my personal portfolio website showcasing my **Full Stack MERN development** skills, projects, and experience.

The website is built using:

- **React** for the frontend
- **Webpack** for bundling
- **CSS** for styling
- **Public folder** for images and downloadable CV

## Features

- Responsive design (desktop + mobile)
- Projects, Skills, and Experience sections
- Clickable **Download CV** button
- GitHub link integration
- Smooth dropdown menu for mobile

## Development

### Install dependencies and run project

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

Output will be in the dist/ folder, ready to deploy.
